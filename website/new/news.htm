3rd March 2018 - Version 0.2.0

The WebPagesLayout program uses the latest update to create its own website. The website can be found at https://nickmat.github.io/WebPageLayout/website/index.htm

This version now outputs the files that are common to all the sister websites, namely, the livery.css and common graphic files and so provides a central place to maintain and refine them. A common naming convention for the icon and bullet images has also been introduced. These steps will make it easier to use with TheFamilyPack website, which will happen soon.
