Introduction

WebPageLayout (wpl) is a program used to create and maintain the websites of a small family of related projects. These projects are:- HistoryCal, HistoryLoc and TheFamilyPack.

This website serves as an example and test site as well as providing useful documentation on how the program is used.

Note that there will be no formal release of this program, since its only intended use is in maintaining the above websites. However, if there is someone out there who really wants to create a website with the look and feel that this program produces, then please contact the author.
