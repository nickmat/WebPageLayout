In Development

The program is developed on an as-needed bases. Any media or file that is required by more than one of the client websites can be included.

The current program is only concerned with the webpages template and the standard menus. No consideration is given to the content - other than to copy it between versions. A future version may check the content for valid relative file links.

The doctype for the pages will be changed to HTML5 at some point.

A Blog Section

Consideration is being given to adding a blog section. The blogs would be written using markdown and converted to html.

First thoughts, a directory webblog would be used to write the blogs. For these pages, instead of copying the html content to the updated file, it would transfer converted markdown content.

Possible conversion library is github.com/progsource/maddy.
